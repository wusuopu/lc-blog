Linux下mame游戏模拟器

Jun 10th, 2010 12:00 am | Comments

Linux下mame游戏模拟器有xmame、sdlmame。

而由于XMAME已经停止开发所以想在LINUX下玩到最新的MAME游戏只能选择SDLMAME了。

执行命令：sudo apt-get install sdlmame 安装sdlmame。

执行命令： sdlmame roms文件名 开始游戏，也可在图形界面下操作。

可以到这个网站下载roms文件 http://www.rom-world.com/

GameConqueror——Linux下的游戏作弊器

May 30th, 2010 12:00 am | Comments

GameConqueror 是一款 Linux 游戏修改工具，用 PyGTK 写成，以 scanmem 作为后端。

现在已经实现了基本的搜索功能，支持不同的数据类型和搜索类型。

数据类型：

不同长度的类型：int{8/16/32/64}、float{32/64}

未知长度的类型：int、float

未知类型：number

字节串和字符串：bytearray、string

搜索类型：相等、大于、小于、变化、未变、增大、减少

运行需要 Python 和 Python-GTK2，编译需要 libreadline。

安装带图形界面的编译:

./configure –enable-gui && make sudo make install

安装不带图形界面的编译:

./configure && make sudo make install

安装完后在终端运行 gameconqueror，启动界面方式；运行 scanmem，启动命令行方式。还有不懂的自己看帮助。

主页：http://code.google.com/p/scanmem/
