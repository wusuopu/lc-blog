Riot.js之初体验

Riot(http://riotjs.com/)按照官方的介绍，它是一个类似于 React 的微型框架。
压缩之后的文件只有差不多 15K 的大小，相比其他基本上都是上百K大小的框架来说确实是很微型的。

同时它的官方还给出了与 React 和 Polymer 的对比，各位感兴趣可以看看： http://riotjs.com/compare/

下面通过一个例子来体验一下。
