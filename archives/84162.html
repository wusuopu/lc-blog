这里有个麻烦的地方就是，当我在真机设备上调试时。每次都需要执行 ifconfig 命令，然后将 localhost 修改为我的 ip 地址。并且在使用 git 进行代码管理时，一不小心将修改后的文件提交上去了，其他同事在 pull 时又会与自己的冲突。
